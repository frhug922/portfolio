### ✅ 구현 내용

### 1) Google Sheets 기반 데이터 테이블 로드 시스템

- 기능 설명: Google Sheets API를 활용하여 빌드 재배포 없이, 즉 시트 내 데이터 수정만으로 게임 내 데이터를 업데이트 할 수 있는 시스템입니다.
- 구현 내용: Google Sheets CSV export URL을 통해 데이터를 불러오고, 이를 파싱하여 게임 내 데이터 테이블로 변환하는 기능을 구현했습니다. UnityWebRequest를 사용하여 비동기적으로 데이터를 가져오고, JSONUtility를 활용하여 데이터를 구조화하였습니다.
- 선택 이유: 기획자가 직접 데이터 수정을 할 수 있도록 하고, 즉시 반영되는 시스템을 구축하여 빠른 피드백과 반복 작업이 가능하도록 하기 위해 선택했습니다.
- 결과: 밸런스 조정 및 콘텐츠 업데이트가 신속하게 이루어져 개발 효율성이 크게 향상되었습니다.

### 2) 밤/낮 주기 시스템

- 기능 설명: 게임 내 시간을 낮과 밤으로 구분하여, 각 시간대에 따라 시야가 제한되고 플레이어의 행동에 영향을 미치는 시스템입니다.
- 구현 내용: DayNightCycle 스크립트를 작성하여, 일정 시간 간격으로 낮과 밤을 전환하는 기능을 구현했습니다. 또한, 조명 효과와 시야 제한 기능을 추가하여 밤에는 시야가 좁아지고, 낮에는 넓어지는 효과를 적용했습니다.
- 선택 이유: 시간대에 따른 게임 플레이 변화를 통해 전략적인 요소를 추가하고, 플레이어에게 다양한 경험을 제공하기 위해 선택했습니다.
  - 결과: 낮/밤 주기 시스템이 게임의 몰입감을 높이고, 플레이어가 시간에 따라 전략을 변경하는 재미를 느낄 수 있게 되었습니다.
