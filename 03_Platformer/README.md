### ✅ 구현 내용

### 1) List 기반 UI 시스템

- 기능 설명: 여러가지 UI 패널들을 List 자료구조를 활용하여 관리하는 시스템입니다.
- 구현 내용: MainSceneUIController 스크립트에서 SetShow()된 UI 패널들을 List로 관리하고, 필요에 따라 활성화/비활성화하는 방식을 구현했습니다.
- 선택 이유: Stack 자료구조는 후입선출(LIFO) 방식으로 UI 패널을 관리하는 데 적합하지만, 수평적인 UI 패널 전환이 필요하기 때문에 List 자료구조를 선택했습니다.
- 결과: 여러 UI 패널들을 Stack처럼 쌓아서 관리를 하면서도 수평적인 전환이 가능해져, 여러 UI 패널들을 연 상태에서도 좌우 화면전환을 통해 여러 UI를 동시에 확인할 수 있게 되었습니다.

### 2) Photon PUN2 기반 멀티플레이어 시스템

- 기능 설명: 여러 사용자가 동시에 접속하여 플레이할 수 있는 멀티플레이어 시스템입니다.
- 구현 내용: 방을 단순히 생성, 입장하는 기능부터, 플레이어 닉네임 설정, 스킨 설정 등 게임 플레이에 필요한 다양한 기능들을 Photon PUN2를 활용하여 구현했습니다.
- 선택 이유: Photon PUN2는 Unity 환경에 최적화된 실시간 네트워크 솔루션으로, 서버 구축 없이도 손쉽게 멀티플레이를 구현할 수 있습니다.
- 결과: 안정적인 멀티플레이어 환경을 제공하여, 여러 사용자가 동시에 접속하여 원활하게 게임을 즐길 수 있게 되었습니다.
